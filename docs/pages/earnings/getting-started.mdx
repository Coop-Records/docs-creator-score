# Coop Earnings API Reference

Retrieve a creator's onchain earnings and related metrics. This API provides metrics such as total creator revenue and unique minters/collectors.

> **Note:** Replace `[address]` with the creator's Ethereum address.

## Endpoint

```http
GET https://cooprecords.xyz/api/earnings?address=[address]
```

## Parameters

| Name    | Type   | Required | Description                     |
| ------- | ------ | -------- | ------------------------------- |
| address | string | Yes      | Ethereum address of the creator |

## Request Examples

:::code-group

```bash [cURL]
curl -X GET "https://cooprecords.xyz/api/earnings?address=0xbCefc4906b443e4DB64E2b00b9af2C39e76c785c"
```

```python [Python]
import requests

def get_earnings(address: str):
    url = f"https://cooprecords.xyz/api/earnings"
    params = {"address": address}
    response = requests.get(url, params=params)
    response.raise_for_status()
    return response.json()

# Example usage:
# get_earnings("0xbCefc4906b443e4DB64E2b00b9af2C39e76c785c")
```

```javascript [JavaScript]
async function getEarnings(address) {
  const response = await fetch(
    `https://cooprecords.xyz/api/earnings?address=${address}`
  );
  if (!response.ok) throw new Error(`HTTP error! status: ${response.status}`);
  return await response.json();
}
// Example usage:
// getEarnings("0xbCefc4906b443e4DB64E2b00b9af2C39e76c785c")
```

```typescript [TypeScript]
export interface EarningsResponse {
  earnings: number; // in ETH
  earningsUsd: string;
  collectorsCount: number;
}

async function getEarnings(address: string): Promise<EarningsResponse> {
  const response = await fetch(
    `https://cooprecords.xyz/api/earnings?address=${address}`
  );
  if (!response.ok) throw new Error(`HTTP error! status: ${response.status}`);
  return await response.json();
}
```

:::

## Response Format

The API returns JSON responses. Here's an example success response:

```json
{
  "earnings": 7.336290909005999,
  "earningsUsd": "18855.37",
  "collectorsCount": 13816
}
```

## Response Properties

| Property        | Type   | Description                         |
| --------------- | ------ | ----------------------------------- |
| earnings        | number | Total creator revenue in ETH        |
| earningsUsd     | string | Total creator revenue in USD        |
| collectorsCount | number | Number of unique collectors/minters |

## Error Response

If an error occurs, the API returns:

```json
{
  "error": "Invalid or missing address parameter."
}
```
