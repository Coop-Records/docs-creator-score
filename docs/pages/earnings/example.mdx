# Example Requests: Coop Earnings API

Below are example requests to fetch the Coop Earnings for specific creators in multiple languages:

## Daniel Allan

**Address:** `0xbCefc4906b443e4DB64E2b00b9af2C39e76c785c`

:::code-group

```bash [cURL]
curl -X GET "https://cooprecords.xyz/api/earnings?address=0xbCefc4906b443e4DB64E2b00b9af2C39e76c785c"
```

```python [Python]
import requests

def get_earnings():
    url = "https://cooprecords.xyz/api/earnings"
    params = {"address": "0xbCefc4906b443e4DB64E2b00b9af2C39e76c785c"}
    response = requests.get(url, params=params)
    response.raise_for_status()
    return response.json()

# print(get_earnings())
```

```javascript [JavaScript]
async function getEarnings() {
  const response = await fetch(
    "https://cooprecords.xyz/api/earnings?address=0xbCefc4906b443e4DB64E2b00b9af2C39e76c785c"
  );
  if (!response.ok) throw new Error(`HTTP error! status: ${response.status}`);
  return await response.json();
}
// getEarnings().then(console.log);
```
